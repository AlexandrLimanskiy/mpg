VPI Discord Bot - функциональность проекта

Этот проект представляет собой игровой Discord-бот для военно-политической игры. Ниже приведено краткое описание текущих возможностей, которое поможет в дальнейшем проектировать новые механики.

== Основные возможности ==
1. **Выбор страны**
   - Команда `/select_country` позволяет выбрать одну из доступных стран (Украина, Россия, Беларусь).
   - `/my_country` выводит подробную информацию о выбранной стране, её бюджет, армию и регионы.
2. **Политические системы**
   - Список систем можно посмотреть через `/political_systems`.
   - Администратор может установить систему командой `/admin_set_political_system`.
   - Эффекты систем влияют на экономику, производство и боевую мощь.
3. **Экономика и производство**
   - Игрок имеет бюджет и может строить военные заводы пехотного вооружения командой `/build_factory`.
   - Производство оружия отображается в `/factories` и учитывает бонусы политической системы.
   - Покупка техники производится через `/buy` или отдельные команды (`/buy_tank`, `/buy_bomber` и т.д.).
   - Инвентарь и доступное вооружение отображаются командой `/inventory`.
4. **Боевые действия**
   - Команда `/attack` позволяет атаковать другого игрока, учитывая случайный фактор и потери.
   - История последних сражений доступна через `/history`.
5. **Регионы стран**
   - При выборе страны игроку автоматически назначаются её регионы.
   - `/regions` и `/view_region` выводят информацию о состоянии регионов и их возможных бонусах.
   - Администратор может изменять контроль и повреждения регионов (`/admin_region_control`, `/admin_damage_region`).
6. **Административные команды**
   - Выдача и установка денег и вооружения (`/grant_money`, `/set_money`, `/grant_ammo`).
   - Назначение и сброс страны игроку (`/set_country`, `/reset_country`).
   - Вызов справки по админ-командам `/admin_help`.
7. **Другие функции**
   - Команда `/help_vpi` показывает справку по основным разделам игры.
   - Проект использует базу данных SQLite (`vpi.db`) для хранения игроков, инвентаря, заводов, истории боёв и регионов.

== Возможности для дальнейшего развития ==
- Реализация системы построек в регионах, которая будет активировать их бонусы и давать экономические или военные преимущества.
- Добавление новых стран и регионов.
- Расширение списка юнитов и улучшение баланса боевой системы.
- Разработка событий, влияющих на политические системы и статус регионов.
